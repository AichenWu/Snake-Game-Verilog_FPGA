# Snake Game Verilog/FPGA
修改他人專案: https://github.com/vale5230/Snake-Game-in-FPGA  
原專案功能:  
(1) 在8x8的LED全彩點矩陣上，隨機生成蛇的食物(會避開蛇的身體位置)  
(2) 每吃到一點得一分，16-bits LED會多亮一個燈  
(3) 暫停功能  
(4) 超過8分就將蛇的身體變色  

新增/修改功能:  
(1) 超過4分就將蛇的身體變色  
(2) 完成7分就贏了，8x8的LED全彩點矩陣顯示笑臉
